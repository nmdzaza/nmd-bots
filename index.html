Before You Start

⚙️ Setup prerequisites

Every bot runs on Python. Do this once and all 7 bots are ready to go.

1

Get your bot token from BotFather

Open Telegram, search @BotFather, send /newbot, follow the prompts. Copy the token it gives you — looks like 123456789:AAF...

t.me/BotFather → /newbot → copy token
2

Install Python dependencies

Each bot has a requirements.txt. Install everything in one command.

pip install python-telegram-bot python-dotenv httpx groq
3

Create your .env file

Drop a .env file in the same folder as the bot. Fill in your tokens. Each bot below shows exactly which keys it needs.

~/.env — shared across all bots
BOT_TOKEN="your_bot_token_here"
ADMIN_ID="your_telegram_user_id" # get from @userinfobot
GROQ_API_KEY="gsk_..." # free at console.groq.com
CHANNEL_ID="-1001234567890" # your channel's chat ID
4

Run the bot

Each bot is a single Python file. Run it directly. Add it to pm2 or a systemd service to keep it alive 24/7.

python3 bot_name.py
pm2 start bot_name.py --name my-bot # keep alive
